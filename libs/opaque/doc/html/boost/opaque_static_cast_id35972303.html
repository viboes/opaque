Function template opaque_static_cast

boost::opaque_static_cast — Conversion from a new_class<> to a type convertible from UT.

Synopsis

// In header: <boost/opaque/new_class.hpp>


template<typename T, typename Final, typename UT, typename MetaMixinSeq, 
         typename Base> 
  T opaque_static_cast(opaque::new_class< Final, UT, MetaMixinSeq, Base > const & v);

Description

Requires:

  • Final must publicly inherit from opaque::new_class<Final, UT, MetaMixinSeq, Base>.

  • UT must be CopyConstructible and Assignable.

  • MetaMixinSeq must be a model of MetaMixinSeq.

  • UT must be convertible to T.

  • Base must inherit directly or indirectly from base_opaque_type or be base_opaque_type.
