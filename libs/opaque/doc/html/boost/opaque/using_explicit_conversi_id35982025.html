Description

mixin generating the explicit conversion to all the underlying types hierarchy. This meta-mixin provides the Final class explicit conversions for all the underlying types hierarchy (when the compiler supports explicit conversion operators). For portability purposed the library provide as workaround a convert_to non member function which follows the Boost.conversion protocol.
