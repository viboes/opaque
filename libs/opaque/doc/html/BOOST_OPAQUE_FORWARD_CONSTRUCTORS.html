Macro BOOST_OPAQUE_FORWARD_CONSTRUCTORS

BOOST_OPAQUE_FORWARD_CONSTRUCTORS

Synopsis

// In header: <boost/opaque/macros/forward_constructors.hpp>

BOOST_OPAQUE_FORWARD_CONSTRUCTORS(OT, BASE)

Description

forward the constructors from the new class OT to the base class @ BASE

Note(s): This macro can be used only inside the public part of a class.

Parameters:

- OT: the class been defined.

- BASE: the base class.

Result: 

    OT(){}
    template <typename W>
    explicit OT(W w)
        : BASE(w)
    {}
    OT(OT const& r)
        : BASE(r.val_)
    {}
