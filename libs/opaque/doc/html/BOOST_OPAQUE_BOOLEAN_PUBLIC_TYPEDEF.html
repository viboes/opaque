Macro BOOST_OPAQUE_BOOLEAN_PUBLIC_TYPEDEF

BOOST_OPAQUE_BOOLEAN_PUBLIC_TYPEDEF

Synopsis

// In header: <boost/opaque/macros.hpp>

BOOST_OPAQUE_BOOLEAN_PUBLIC_TYPEDEF(UT, OT)

Description

Declares a public_opaque_type typedef OT defining a specific tag OT_tag having boost::opaque::boolean as Boolean type.

Note(s): This macro must be used inside the OT's namespace.

Parameters:

- UT: the underlying type.

- OT: the opaque type.

Result: 

 struct OT_tag {};
 typedef boost::opaque::public_opaque_type<UT, OT_tag, boost::opaque::boolean> OT
