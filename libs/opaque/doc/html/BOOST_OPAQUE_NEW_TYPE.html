Macro BOOST_OPAQUE_NEW_TYPE

BOOST_OPAQUE_NEW_TYPE — Generates the code to define a new typedef.

Synopsis

// In header: <boost/opaque/new_type_macro.hpp>

BOOST_OPAQUE_NEW_TYPE(UT, NT, MMSEQ)

Description

Parameters:

- UT: the underlying type.

- NT: the new type to be defined .

- MMSEQ: the META_MIXIN_SEQ sequence.

Result: If MMSEQ is ( (MM1) ... ((MMn) ) 

  struct NC_tag {};
  typedef boost::opaque::new_type< UT, NC_tag,
      boost::mpl::vector<MM1, ..., MMn>
  > NT
